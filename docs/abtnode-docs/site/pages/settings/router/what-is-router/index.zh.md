路由可以方便的让你管理 Blocklet Server 中运行的服务，为每一个 Blocklet 设置一个自己的有命名意义的路由名称，将会让你的 Blocklet 更加方便地被管理。

### 路由设置页面

我们先来看一下路由规则页面长什么样吧。

![router-settings-zh](./images/router-settings-zh.png)

可以看到，在 `路由规则` 栏目下面，我们可以为已经运行的 Blocklet 服务自由设置路由规则。
