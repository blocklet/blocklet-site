# Blocklet Sites

This repo include website and documentation for blocklet framework and blocklet developers.

## Contribution

```bash
git clone git@github.com:blocklet/blocklet-site.git
cd blocklet-site
make dep
```

### Blocklet Site

```bash
cd website/pages
blocklet dev
```

Doc site:

```bash
cd website/docs
npm run dev:child
```

### Developer Site

```bash
cd developer/pages
blocklet dev
```

Doc site:

```bash
cd developer/docs
npm run dev:child
```
