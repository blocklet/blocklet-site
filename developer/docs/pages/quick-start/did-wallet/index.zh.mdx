## 为什么我需要 DID 钱包？

[DID Wallet](https://didwallet.io)是一个去中心化的应用程序，用来管理数字资产（如[token](https://chain.abtnet.io/docs/en/conceptual/tokens)和[NFT](https://chain.abtnet.io/docs/en/conceptual/nfts)）和[去中心化的身份](https://chain.abtnet.io/docs/en/conceptual/did)。

DID Wallet 在区块链开发工作流程中需要完成以下任务。

- 在与区块小程序对话时，管理你的去中心化身份
- 接收并出示区块链服务器护照
- 接收并出示区块链所有者护照
- 接收和花费区块链代币，NFTs
- 授权区块链和服务器的其他危险操作

## DID 钱包所支持的平台？

DID Wallet 目前完全支持以下平台。

- [安卓](https://play.google.com/store/apps/details?id=com.arcblock.wallet.app.product)
- [iOS](https://apps.apple.com/app/id1460083542)
- [浏览器 WebApp](https://web.abtwallet.io/)，仅支持现代浏览器，如 Chrome、Safari、Edge 和 Firefox
- [浏览器扩展](https://chrome.google.com/webstore/detail/did-wallet/ibjflpbmadchofnbpppegdbnifdgincp)，目前仅支持 Chrome 版本

WEB 端的 DID 钱包通常被开发人员青睐，因为他们提供了一个更短的反馈循环。

## 如何设置 DID 钱包？

有两种方法来设置 DID 钱包。

- [创建新钱包](https://support.didwallet.io/en/howToInitWallet)
- [从备份中恢复](https://support.didwallet.io/en/howToRecoverWallet)

设置钱包后，你将得到 3 个默认账户，每当你使用一个新的区块链时，DID 钱包将生成一个新的 DID，更多信息请参考。[为什么有多个账户？](https://support.didwallet.io/en/whyMultiAddresses)
